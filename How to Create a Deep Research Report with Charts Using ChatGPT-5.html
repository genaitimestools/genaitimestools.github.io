Do you think that ChatGPT's Deep Research function can't use charts? In fact, this Deep Research function can also use charts! In this guide, I explore how to create a report with charts using ChatGPT Deep Research and Canvas functions. Before getting started, you need a free ChatGPT account.

Step-by-Step Guide

Step 1: Create a Deep Research report.

First, please create a report on your topic.

  1. Visit https://chatgpt.com/.
  2. Click the "+" button near the textbox.
    3. Source: https://chatgpt.com
  3. Click "Deep Research."
    4. Source: https://chatgpt.com
  4. Type this prompt template in the textbox. 
    Create a report on (topic). Also, use Chart.js to visualize data related to the topic.
    Source: https://chatgpt.com
  5. Fill the (topic) in the prompt template with a topic you want to know.
  6. Click Enter.
  7. After clicking Enter, you will be asked some clarification questions. Please answer them.
  8. Afterwards, research will be conducted.
  9. After that, execute this prompt. 
    Create the same report in HTML format using the Canvas function.
  10. The HTML report creation will start.
  11. After the creation finishes, scroll down the chat, and click "Preview" in the top right corner of the generated HTML code block.
    Source: https://chatgpt.com
  12. The HTML code will be previewed.
    Source: https://chatgpt.com

Step 2: Download the HTML file

Once the report is created, save the HTML file on your computer.

  1. Click "..." in the top, right navigation bar.
    2. Source: https://chatgpt.com
  2. Click "Download."

Here's the example I created.

Use Cases

Market research report

In market research, you often use many chart types, including bar charts for comparing product preferences and demographics, pie charts for showing market share, and line graphs for tracking sales trends or customer sentiments. In a scenario like this, mention must-include data in this prompt template. Here's the template, example, and result.

Prompt template

Create a report on (topic). Also, use Chart.js to visualize data related to the topic. Please include these data and the chart types to visualize:
- (data A)
- (data B)
- (data C)
- ...

Example

Create a report on the AI coding assistant software market trend in the US in 2025. Also, embed Chart.js code for charts to visualize data related to the topic. Finally, you must include these data in the report:
- growth rate
- usage by developer type
- market share of tools

The result

White paper

A white paper is an authoritative, research-based document that includes data from external and original research and the original analysis. They are mainly used for generating companies' leads. Since a white paper presents information from external sources, you can offload data collection and summarization and chart generation using ChatGPT's Deep Research function. Here's the prompt template and example.

Prompt Template

Since I would like to create a white paper on (the white paper's topic), create a report on (topic). Also, use Chart.js to visualize data related to the topic.

Example

Since I would like to create a white paper on why our AI spreadsheet tool can solve the challenges of the logistics industry, create a report on the challenges of the logistics companies. Also, use Chart.js to visualize data related to the topic. Finally, include these data below in the report:

- how many logistics companies were affected by their challenges. 
- real data that verify the impact, such as demands for real-time tracking capabilities of goods.

Result

Investment Report

In an investment report, you usually use different types of charts. For example,while candlestick charts or OHLC charts are suitable for stock and commodity price analysis, line charts are suitable for analyzing the long-term impact of your investment. If you instruct ChatGPT to combine these charts with text information, you can get valuable insights into the long-term trends. Here's the prompt template and example.

Prompt Template

Create an investment report on (topic). Also, use Chart.js to visualize data related to the topic.

Example

Create an investment report on global semiconductor demand and cite sources for every factual claim. Also, use Chart.js to visualize data related to the topic.

Result

A chart isn't displayed.

When a chart seems to have syntax errors or is not visible, ask ChatGPT below in the same chat you generate the chart.

It seems the chart is not visible. Check the syntax of the code.
